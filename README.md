# Betternship Monorepo

This repository contains a simple full-stack application with a React frontend (configured with Tailwind CSS) and a Node/Express backend.

## Apps

- **frontend**: React application source code and configuration.
- **backend**: Node/Express API server using MongoDB via Mongoose.

## Getting started

From the repository root:

### 1. Install dependencies

```bash
cd frontend && npm install
cd ../backend && npm install
```

### 2. Run the apps

- **Frontend (React + Tailwind)**  
  From the `frontend` directory:

  ```bash
  npm start
  ```

- **Backend (Node/Express)**  
  From the `backend` directory:

  ```bash
  npm run dev
  ```

## Requirements

- Node.js and npm installed on your machine.
